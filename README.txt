Данное приложение - Web API для работы со списком дел. Написано на платформе ASP.NET Core.

Каждый пункт в списке дел имеет:
- уникальный идентификатор;
- название;
- выполнено или нет.

Реализованы все CRUD-операции. 

Пример взят из https://habr.com/ru/companies/microsoft/articles/312878. Доработаны запросы PUT и PATCH:
- PUT-запрос не выдает ошибку при передаче в объект не всех свойств;
- PATCH-запрос не обнуляет не указанные в запросе свойства (метод Update разделен на методы Update и UpdateAll).
